  - x ::

       f ((natural -> rational) -> (natural -> rational))

       f (natural #step
          rational -> rational)
       也就是變換一下參數之間的依賴關係

       注意
       前者 還需要增加一些輔助條件來描述收斂性 等等
       後者 也需要一些額外的描述
