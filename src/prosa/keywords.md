# keywords

2020-09-16

关于程序语言 keyword 的选择，
假设用 keyword 来做前缀，
不同的 keyword 前缀后面可以跟着不同的语法。

## 类型声明与定义

要选取长度一样的英文词，以保持排版优美，
为了这个目的而列表如下：

|            |   |    | let |      | prove | create |         |          |
|            |   |    | val |      | value |        |         |          |
|            |   |    | var |      | build |        |         |          |
