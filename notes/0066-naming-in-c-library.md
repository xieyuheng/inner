C 标准库中，奇怪的带有很多缩写的函数名字，
其实可以帮助程序员突破英语语法的限制。

缩写指向某些临时约定的核心概念，
然后把缩写组成函数名字。

```c
strcpy
strncpy
strcat
strncat
strcmp
strncmp
strlen
strchr
strrchr
```

可以看出，与长名字相比，这些名字完全突破了英语的语法。

问题是这些缩写太临时了，
尤其是其中单个字母的修饰，
比如上面的 `n` 和 `r`。
另外，单个字符所带有的信息量太小了。

这种突破语法限制的命名方式更适合汉语。

汉字可以直接地、形象地锚定不同的核心概念。
然后通过组字为词，创造大量精确的复合词。
复合词之间又由其组成部分之相同而相联系，
使得人们可以猜测新词的意义。
