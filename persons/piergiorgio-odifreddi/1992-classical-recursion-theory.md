# 0 Introduction

在作者的术语里，
classical 意思是研究具体数学结构，
而 modern 意思是研究抽象的数学结构。

这一章介绍了 recursion theory 作为对可计算性的分析，
在各个领域中的应用。

读了这一章，发现我想要了解的，
作为数学基础的构造主义，
就是限制构造方式的构造主义。

比如，限制为用递归函数定义，
或者更进一步，限制为有高效的算法，
可以用来判断集合的成员关系，
以及集合中元素之间的等价。
