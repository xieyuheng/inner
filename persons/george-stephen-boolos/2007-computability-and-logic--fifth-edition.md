# My Motive

[2026-03-19] 在实现 lambda-lisp 时，
想要判断递归定义的函数之间的等价。

但是一般情况下，函数之间的等价是不可判定的，
如何找到一个可判定的子集？

如果找到了，就可以作为 dependent type system 的基础的等价关系。
因此要学习可判定性的基础知识。
