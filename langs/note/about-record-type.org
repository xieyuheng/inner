       一
       作爲文檔
       二
       處理當 produce type 的項非常多的情況
       因爲又名之後 在模式匹配的時候 就能省略不需要的
       三
       在 pattern match 時建議變元的名字

       因爲是 named argument

       也因爲是 named argument
       所以可以在 pattern match 的時候省略
       也就是說
       如果想要加上這個性狀的話
       就要兩個性狀一起加
       並且語法上要有一致性
