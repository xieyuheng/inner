# 货币创造

国家的中央银行可以发行货币，或者说印钱。
但是为了保证整个市场的物价稳定，避免通货膨胀，央行不能随意发行货币。
理想状态下，央行发行新货币的量要与整个社会所创造的新的商品的价值相持平。

国家是通过商业银行的信贷行为来发行货币的。
以石油化工为例，设想某地勘探出了石油资源，
开采出这些石油，必定会给整个社会带来新的商品。
商业银行通过贷款给石油开采公司，以及下游的石油炼化公司，来发行这部分货币。

商业银行贷款给企业的过程，就是发行货币的过程。
「发行货币」并不是真的「印钱」，而是在银行的账户上记一笔账。

实际上，商业银行判断应该发放贷款给哪些企业，
其准则是预期的回报和风险，
而不是因为社会有了新的商品才去创造货币。

这代表国家相信市场中的企业知道创造价值的方式。
国家发行国债的行为刚好相反，不是商业银行借钱给企业，
而是财政部行向整个社会借钱，
这代表国家知道创造价值的方向与方式，
比如基础设施建设。

注意，这里财政部发行国债，是而不是中央银行。
中央银行可能在二级市场买卖国债。
中央银行不能直接向财政部购买国债，
如果允许，就相当于把财政赤字转化为了货币，
大多数国家法律禁止这种行为。

# 货币乘数

中央银行通过设定准备金率，
来调控商业银行的货币创造能力，
并且把发行货币的权利让渡给商业银行。

# 商业银行

银行的资产负债表中，资产 = 负债 + 所有者权益。

资产包括：

- 现金
- 客户贷款（为主要）

负债：

- 客户存款（为主要）

所有者权益（自有资本）：

- 股本：发行股票收到的本金。

资产减少时（如坏账时），所有者权益会减少。

客户存款时：

- 负债 -- 客户存款 增加
- 资产 -- 现金 增加

向客户放贷时（创造货币的过程）：

- 负债 -- 客户存款 增加
- 资产 -- 客户贷款 增加

deepseek：从资产负债表看银行的关键特性

- 高杠杆经营：例如，银行用 80 亿 的自有资本（所有者权益），可以支撑起
  1,110 亿 的总资产，杠杆倍数很高。这放大了收益，也放大了风险。

- 期限转换：负债端多是短期的存款（客户可能随时支取），而资产端多是长期
  的贷款。银行依靠“大数法则”和流动性管理来应对日常兑付。

- 信用风险承担：主要资产（贷款）的价值取决于借款人能否偿还，因此银行的
  核心能力是信用风险评估和管理。

- 受严格监管：为防止银行过度冒险，监管机构设定了多个基于资产负债表的指标，如：

  - 资本充足率 = 合格资本 / 风险加权资产 ≥ 8%（巴塞尔协议）
  - 流动性覆盖率 = 优质流动性资产 / 未来30天净现金流出 ≥ 100%
  - 存贷比 = 贷款总额 / 存款总额（作为流动性监测指标）

deepseek：总结

商业银行的资产负债表是其吸收存款、发放贷款、创造货币、管理风险和赚取利
差等所有活动的财务镜像。资产端体现了银行的资金运用和盈利模式，负债端则
揭示了其资金来源的稳定性与成本。
